========================================
config 配置文件夹说明
========================================

【文件夹作用】
本文件夹用于存放 API 密钥配置与大模型计费规则。

----------------------------------------
文件清单
----------------------------------------

1. settings.yaml    - API 密钥与端点配置
2. cost_rules.json  - 大模型 Token 计费标准

----------------------------------------
settings.yaml 配置说明
----------------------------------------

本文件是核心配置文件，用于连接大模型 API 服务。

配置项说明：

  api:
    endpoint: "https://your-api-endpoint.com/v1"
      # API 服务地址，必须以 /v1 结尾

    api_key: "your_api_key_here"
      # 你的 API 密钥，从服务商后台获取
      # 注意：此文件已被 .gitignore 忽略，不会被提交到 Git

    model: "mimo-v2.5"
      # 必须使用 mimo-v2.5 多模态视觉模型
      # 该模型支持网页截屏分析和视觉理解

  browser:
    headless: true
      # 是否使用无头浏览器模式
      # true = 后台运行，不显示浏览器窗口
      # false = 显示浏览器窗口，用于调试

    timeout: 30000
      # 浏览器操作超时时间（毫秒）

----------------------------------------
cost_rules.json 计费标准说明
----------------------------------------

本文件定义了各模型的 Token 计费标准。

当前配置（mimo-v2.5）：

  输入（Prompt）：¥2.0 / 百万 Token
  输出（Completion）：¥8.0 / 百万 Token

  换算美元（参考汇率 7.15）：
  输入：$0.28 / 百万 Token
  输出：$1.12 / 百万 Token

----------------------------------------
安全提醒
----------------------------------------

- settings.yaml 包含真实 API 密钥，请勿分享或上传到公开仓库
- 本项目的 .gitignore 已配置忽略此文件
- pre-commit 钩子会在提交时自动检测并拦截包含真实密钥的提交
