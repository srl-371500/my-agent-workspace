========================================
03_多模态爬虫与数据蒸馏系统 模块说明
========================================

【模块作用】
本模块是基于 browser-use 和 mimo-v2.5 的无人值守多模态爬虫与数据蒸馏系统。

核心功能：
- 自动打开浏览器访问指定网站
- 使用 AI 视觉模型分析网页内容
- 生成结构化分析报告
- 记录 Token 消耗和成本

----------------------------------------
多模态网页爬虫与蒸馏模块迁移指南
----------------------------------------

若新项目需要自动爬取、知识蒸馏与 Token 计费账单功能，请按以下步骤操作：

1. 将本文件夹下的 config, services, logs, reports 复制到新项目根目录下

2. 在 services/browser_distillation 下用以下命令配置依赖：
   uv sync

3. 在 config/settings.yaml 中填写 mimo-v2.5 密钥：
   api:
     endpoint: "https://token-plan-sgp.xiaomimimo.com/v1"
     api_key: "your-api-key-here"  # 替换为你的 API 密钥
     model: "mimo-v2.5"

4. 手动运行测试：
   cd services/browser_distillation
   uv run python test_minimal.py

5. 手动运行分析：
   双击 run_distillation.bat
   或者运行：cd services/browser_distillation && uv run python analyze_portfolio.py

6. 注册 Windows 定时任务（可选）：
   - 以管理员身份打开 PowerShell
   - cd 到新项目根目录
   - 运行：.\services\browser_distillation\register_cron.ps1

7. 查看消费账单：
   - 审计日志位置：logs/cost_audit.json
   - PowerShell 查看：Get-Content logs\cost_audit.json

----------------------------------------
文件说明
----------------------------------------

config/
  - settings.yaml    - API 密钥与端点配置
  - cost_rules.json  - Token 计费规则

services/browser_distillation/
  - analyze_portfolio.py   - 爬取主脚本
  - cost_audit.py          - 计费模块
  - run_distillation.bat   - 一键手动运行脚本
  - register_cron.ps1      - 定时任务注册脚本
  - test_minimal.py        - API 连通性测试脚本

logs/
  - cost_audit.json        - Token 成本审计日志

reports/
  - portfolio_inspiration_report.md - 分析报告

----------------------------------------
注意事项
----------------------------------------

- 首次运行需要安装 Playwright 浏览器：
  uv run playwright install chromium

- 如果遇到浏览器扩展超时问题，设置环境变量：
  BROWSER_USE_DISABLE_EXTENSIONS=true

- 定时任务需要以管理员身份运行 PowerShell
